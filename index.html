Who's on first?

Never lose track what has been recorded. The scoreboard reflects the current state of the game, giving the user feedback about what was previously entered. Both the pitch count and at-bat numbers can be hidden by tapping on either value.

Score your game

Juiced Analytics

Know everything about the game being recorded, whether its on an individual or team level. The game report gives statistics relating to the currently selected player. Any player's stats can be viewed for the current game by selecting them as the active player. An overview of the whole game is also available, giving information on a pitch to pitch basis. Both reports are able to be exported.
